Undergraduate Course Projects
  • Black-Scholes-Merton Model. This paper covers a short introduction of European options and derives the famous Black-Scholes equation using a PDE heat equation in continuous time. This project is the final paper for the third-year Mathematical Modelling course.
  • A Glimpse of Machine Learning in Finances. This project covers a machine learning algorithm that uses some companies' financial data to determine if a particular company is "healthy" or "not healthy." This is the final project for the fourth-year Machine Learning and Data Mining course. Access the project's summary here.
  • Impact of sustainable goals on social welfare and resilience. This presentation is the first of two presentations of an ongoing project for my Data Visualization class. This project is about defining an index, which can track impacts of policy changes in the welfare system of countries using complex Principal Component Analysis (PCA) tecniques.
  • To replicate the price of a European option using a risky asset and a risk-free asset, we can use a basic portfolio construction approach. The principle is to create a portfolio of the risky asset and the risk-free asset in such a way that it replicates the payoff of the European option at expiration using Monte Carlo simulations of geometric Brownian motion. The python script for this method is (GBM Option Pricing in Python). To view the PDF version this script please view here.
  • Similarly, we can use the Binomial Option Pricing Model (BOPM), to replicate the price of a European option using a risky asset and a risk-free asset. This is the risk-neutral pricing approach. The python script (Binomial Option Pricing in Python) uses Binomial model to price European options (PDF version: Binomial Option Pricing In Python - PDF).
Graduate Course Projects
  • The python script (Crank-Nicolson Option Pricing in Python) uses Crank-Nicolson finite difference method to evalaute price of European call option, with risk free interest rate of 5%, volatility of 30%, stricke price of $100 and initial price of the underlying asset as $80. (PDF version with plot: Crank-Nicolson Option Pricing In Python - PDF). The MATLAB version of the same method is attached as PDf version (Crank-Nicolson Option Pricing in MATLAB - PDF). We can see that the same model can be constructed with simpler code structure, giving us the same results. The Black-Scholes conditions are considered to write these scripts.

  • My Master's research project centered on pioneering Loss Layering model, a transformative framework for insurers to strategically manage risk exposure by retaining specific loss layers while transferring higher layers. Our innovative approach involves classifying regional losses into "near maximum" and "near minimum" values, followed by precise layer allocation using non-linear regression. The comprehensive integration of all loss values, not just tail distributions, sets our work apart, providing insurers with a nuanced risk perspective. This framework empowers insurers to strategically allocate assets, diversify risk, and enhance coverage for catastrophic perils, ensuring adaptable responses to evolving climate patterns. Ultimately, my research contributes to reshaping risk management strategies and fortifying insurers' resilience in the face of unprecedented events. Please see my presentation slides for this research project: Risk Layering - A Loss Classification Approach
