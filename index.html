Projects

Course Projects
  • Black-Scholes-Merton Model. This paper covers a short introduction of European options and derives the famous Black-Scholes equation using a PDE heat equation in continuous time. This project is the final paper for the third-year Mathematical Modelling course.
  • A Glimpse of Machine Learning in Finances. This project covers a machine learning algorithm that uses some companies' financial data to determine if a particular company is "healthy" or "not healthy." This is the final project for the fourth-year Machine Learning and Data Mining course. Access the project's summary here.
  • Impact of sustainable goals on social welfare and resilience. This presentation is the first of two presentations of an ongoing project for my Data Visualization class. This project is about defining an index, which can track impacts of policy changes in the welfare system of countries using complex Principal Component Analysis (PCA) tecniques.
